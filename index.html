Public Legal Record System (PLRS)

A public, immutable, independently verifiable legal record system
PLRS is not a database.
It is a cryptographic public ledger for legal and protective records that replaces silent editing, closed reporting pipelines, and untraceable revisions.
Important: This site is only a mirror. You are not required to trust this website or any institution that hosts it. All records are independently verifiable using open tools and published hashes.

What PLRS Does

  • Creates immutable legal Event records
  • Allows formal public Challenges without overwriting history
  • Prevents silent edits, backdating, and report laundering
  • Enables court-grade cryptographic verification
  • Allows independent public auditing without permission

Core Public Files

  • Event Index (public_index.json)
  • Challenge Index (public_challenge_index.json)
  • Event Records (E###.json)
  • Challenge Records (C###.json)

Verification

Anyone can independently prove whether a record has been altered. No accounts. No permission. No trust required.

What PLRS Does Not Do

  • It does not determine guilt or innocence
  • It does not replace judges or courts
  • It does not make legal judgments
  • It does not expose identities without consent

Why This Exists

Traditional reporting systems allow:
  • Silent revisions
  • Timeline manipulation
  • Untraceable report changes
  • Closed review pipelines

PLRS replaces that with:
  • Append-only public records
  • Cryptographic verification
  • Permanent adversarial history
  • Independent auditability
